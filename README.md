# DirectCall

P2P голосовые звонки для ПК — без аккаунтов, без пинга, подключение по коду.

Один жмёт **«Создать звонок»** и получает код типа `483-291`. Второй вводит код — и вы в разговоре. Сервер участвует только в рукопожатии (2 секунды), дальше звук идёт напрямую между компьютерами (WebRTC: Opus, эхоподавление, шумодав).

## Возможности

- 🎙 Голос 1-на-1 с минимально возможной задержкой (чистый P2P)
- 🔢 Подключение по 6-значному коду — из любой сети, без настроек роутера
- 🖥 Демонстрация экрана или отдельного окна (со звуком системы)
- 💬 Чат прямо в звонке (тоже P2P, через DataChannel)
- 🔇 Переключаемое шумоподавление
- 📶 Пинг и длительность звонка на экране

## Установка

Скачай `DirectCall-Setup-x.x.x.exe` из [Releases](../../releases), запусти — появится ярлык на рабочем столе. SmartScreen при первом запуске: «Подробнее» → «Выполнить в любом случае».

## Сборка из исходников

```
npm install
npm start        # запуск в dev-режиме
npm run dist     # сборка установщика (release/)
```
