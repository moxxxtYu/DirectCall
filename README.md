# DirectCall

P2P голосовые звонки для ПК — без серверов, без аккаунтов, без пинга.

Один жмёт **«Создать звонок»** — его ПК становится точкой подключения. Второй вводит его IP — и вы в разговоре. Звук идёт напрямую между компьютерами (WebRTC: Opus, эхоподавление, шумодав).

## Возможности

- 🎙 Голос 1-на-1 с минимально возможной задержкой (чистый P2P)
- 🖥 Демонстрация экрана или отдельного окна (со звуком системы)
- 💬 Чат прямо в звонке (тоже P2P, через DataChannel)
- 🔇 Переключаемое шумоподавление
- 📡 Твои IP-адреса на главном экране — скинул другу и всё

## Установка

Скачай `DirectCall-Setup-x.x.x.exe` из [Releases](../../releases), запусти — появится ярлык на рабочем столе.

## Как позвонить

- **Одна сеть / общий VPN** — работает сразу: хост жмёт «Создать звонок», друг вводит его IP.
- **Через интернет** — тому, кто хостит, нужно пробросить порт **9944** (TCP) на роутере, друг подключается на внешний IP.
- При первом хостинге Windows спросит разрешение в брандмауэре — разрешить.

## Сборка из исходников
